The Philosophy of Ruby

The progamming language Ruby is widely used in web development (thanks to Ruby on Rails). However, as a universal purpose language, its usage is not limited to the web. This language itself has many interesting features that are designed with so called philosophy of Ruby.

Features

Everything is an object

In Ruby, everything you see, you can operate is actually an object. Not only a string, an integer, a class itself and even the scope you run your main script is also an object.
